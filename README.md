# go-openspec
A hello world for Spec-Driven Development using [openspec](https://github.com/Fission-AI/OpenSpec) with claude.

## Basic steps

```sh
/opsx:propose Build a command line hello-world program to say hello arg where arg is a command line arg
```

The following workflow markdowns will be generated:  
| Artifact | Purpose |
|----------|---------|
| `proposal.md` | The "why" and "what" - captures intent, scope, and approach |
| `specs/` | Delta specs showing ADDED/MODIFIED/REMOVED requirements |
| `design.md` | The "how" - technical approach and architecture decisions |
| `tasks.md` | Implementation checklist with checkboxes |
