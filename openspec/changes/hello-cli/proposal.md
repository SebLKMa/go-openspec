## Why

We need a simple command-line program that greets a user by name. This serves as the project's initial executable — a hello-world starting point.

## What Changes

- Add a CLI program that accepts a single command-line argument and prints "hello <arg>"
- The program should be invokable from the terminal with a name argument

## Capabilities

### New Capabilities
- `hello-command`: CLI command that accepts a name argument and prints a greeting

### Modified Capabilities
None.

## Impact

- New executable entry point added to the project
- No existing code affected (greenfield)
